# Topic Diversity H-index Analysis

## Project Overview
This project aims to analyze the topic diversity of researchers based on their published works. By utilizing the H-index metric, it provides insights into how diverse and impactful a researcher's contributions are across different fields.

## Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Shibakennn/topic-diversity-hindex-analysis.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd topic-diversity-hindex-analysis
   ```
3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the analysis script**:
   ```bash
   python analyze.py
   ```
5. **View results**:
   The results will be saved in the `results` directory. Check the README for more detailed output descriptions.
